Обзор: Клиентская чать для автоматизированного управления контента для add-reality, написана на фреймворке next.js

Узлы:

Вся система имеет 4 узла:

1) next.js-клиент
(веб-интерфейс для управления файлами и расписанием вручную, создания новых точек вещания)

2) nest.js-сервер
(сервер для обработки запросов с клиента, работы с файлами, генерации xml шаблонов, наблюдения за excel-таблицами)

3) add-reality
(digital signage решение для вещания корпоративного телевидения)

4) excel-файлы
(файл c расписанием ) //пока неясно, будут они в гугле или где-то хоститься
